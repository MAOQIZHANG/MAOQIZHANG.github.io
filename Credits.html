Key Contributions

In this project, we create an interactive way for readers to explore Cournot Competition and its variation.


Our key contribution can be summarized as follows:
1.
2. Model the cournot competition as a prisoner's dilemma by assumption of integer quantity and conversative convention.
3. An interactive and visualized game board for repeated Cournot game (With real-time game result)
4. A Sandbox Mode that allows player to choose Player with different strategy to compete.
5. A calculator that can find out the solution under different condition with input
